# Paradise Stay Frontend

Alternative frontend implementation for the Paradise Hotel coursework project.

## Included functionality
- Home and About pages
- Register and Login
- Room search with filters
- Create reservation
- Cancel reservation
- My bookings page
- Admin overview page
- Admin bookings page
- Admin destinations page
- Protected user and admin routes
- Error handling and validation
- Component tests

## Tech stack
- React
- TypeScript
- React Router
- Axios
- Recharts
- Vitest
- React Testing Library

## How to run
```bash
npm install
npm run dev
```

## Tests
```bash
npm test
npm run coverage
```

## Backend
This frontend uses the provided Paradise Hotel backend API and expects it by default at `http://localhost:4000`.
